### Repository Structure

-   **/chapters**: Contains individual scripts and notebooks for each chapter. The file names correspond to the chapter and topic (e.g., `ch07_markov_switching_model.py`).
-   **/utils**: Contains reusable utility functions (e.g., `nonlinear_tools.py`) that are imported by various chapter scripts.
-   **/data**: Contains any sample CSV data files used in the book.
-   `requirements.txt`: A list of all the Python libraries needed to run the code.
